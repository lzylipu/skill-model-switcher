# 🔀 Model Switcher / 智能大模型切换器

> 🌐 English | [简体中文](./README.md)
> **Deprecated**: Built for legacy OpenClaw. For Hermes Agent, use provider/profile configuration to switch models.
> **已弃用**：此技能为 OpenClaw 构建。Hermes Agent 请直接使用 provider/profile 配置切换模型。

根据任务类型自动切换 DeepSeek V4-Flash（日常）和 V4-Pro（深度推理）。

## ✨ 功能 / Features

- ⚡ **日常模式 / Daily** — V4-Flash 快速响应 / fast lightweight responses
- 🧠 **深度模式 / Reasoning** — V4-Pro 深度推理 / deep inference and analysis
- 🤖 **自动检测 / Auto-detect** — 根据任务复杂度自动切换 / switches automatically based on task complexity
