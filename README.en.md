# Model Switcher

**[English](./README.en.md) | [中文](./README.md)**

> **Legacy**: This was built for OpenClaw. For Hermes Agent, use provider/profile configuration instead.

Auto-switch between DeepSeek V4-Flash (daily) and V4-Pro (deep reasoning) based on task type.

## Features

- **Daily Mode**: V4-Flash for fast responses
- **Deep Mode**: V4-Pro for in-depth reasoning
- **Auto-Detect**: Switch automatically based on task complexity
